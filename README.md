# Features

入力された英数字(A~Z,0~9)や記号(.)を(欧文)モールス信号に変換する機能を有したデバイスドライバーである。

# Usage

DEMOの実行方法など、"hoge"の基本的な使い方を説明する

```
$make
$sudo insmod myled.ko
$sudo chmod 666 /dev/myled0
$echo 変換元の文字 > /dev/myled0
```
